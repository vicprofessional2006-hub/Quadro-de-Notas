# Quadros – Canvas de Notas

Aplicação simples de quadros com canvas infinito e notas autoadesivas.

## Funcionalidades
- Criar quadros
- Canvas infinito com pan
- Duplo clique cria notas
- Autosave local (IndexedDB)
- Tema dark profissional

## Como hospedar no GitHub Pages
1. Crie um repositório
2. Suba todos os arquivos
3. Ative GitHub Pages (branch main / root)
4. Acesse pelo link gerado

Os dados ficam salvos no navegador.
